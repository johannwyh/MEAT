<h1>MEAT: Multiview Diffusion Model for Human Generation on Megapixels with Mesh Attention</h1>

<h4>TL;DR</h4>
<h5>With mesh attention designed for efficient cross-view feature fusion, MEAT is the first human <br> 
multiview diffusion model that can generate dense, view-consistent multiview images at a resolution of 1024x1024.</h5>
